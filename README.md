# ShapeShift-Skeleton

To Run: </br>
  git clone https://github.com/mdivani/shapeshift.git </br>
  cd ShapeShift </br>
  npm install </br>
  npm run dev-server //runs on localhost:8080</br>
  node dev-server.js //runs on localhost:3000</br>

  live version deployed on https://mdivani.github.io/shapeshift/
